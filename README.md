# GameJamStopKillingGames

A Unity-based turn-based card game developed for a game jam with the theme "Stop Killing Games". This project implements strategic card combat mechanics with a narrative-driven experience.

## Quick Overview

This Unity 2022.3.31f1 project features:
- **Turn-based card combat** with strategic gameplay
- **State machine architecture** for game flow management
- **ScriptableObject-driven design** for data configuration
- **Component-based entity system** for players and enemies
- **Modular snap zone system** for card deployment
- **Integrated dialogue system** using Fungus
- **Professional audio management** with spatial sound

## Key Systems

- **Game Management**: Centralized managers with singleton pattern
- **Card System**: Abstract card base with ability composition
- **Entity Framework**: Player/Enemy inheritance with data-driven behaviors
- **State Management**: Comprehensive game state transitions
- **UI Integration**: Reactive interface with game state
- **Audio System**: Dynamic audio management and mixing
